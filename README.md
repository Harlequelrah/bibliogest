# Description

Bibliogest est un projet simple de gestion de bibliothèque .

## Installation

- **Avec Github :**
  ```bash
  git clone https://github.com/Harlequelrah/bibliogest
  ```

## Utilisation

```bash
   cd bibliogest
  elrahapi run
```

## `Technologies et bibliothèques`

- Framework FASTAPI
- Bibliothèque ou Framework elrahapi
